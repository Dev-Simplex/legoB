## Story

**As a** developer,
**I want** a Vite + React + TypeScript scaffold with linting, formatting, and basic CI,
**so that** I can iterate quickly with safe defaults across the LegoB codebase.

## Context

This is the foundational story. It replaces the Phase 0 placeholder `package.json` with a full scaffold, establishes the dev workflow (lint/typecheck/test/build), wires up GitHub Actions, and resolves PO Must-Fix **#M2** (`.env.example`). No functional features yet ‚Äî just the plumbing the rest of the epics depend on.

The repo already has:
- `.aiox/config.yaml`
- `.gitignore` (covers `node_modules`, `dist`, `.env*`, etc.)
- `README.md` (draft)
- `docs/` (planning artifacts ‚Äî shouldn't be touched)
- `.github/workflows/` (empty ‚Äî this story populates it)

## Acceptance Criteria

1. **Vite scaffold installed** ‚Äî Running `npm create vite@latest . -- --template react-ts` semantics produces:
   - `package.json` with React 18.3+, React DOM, TypeScript 5.4+, Vite 5.x as dependencies.
   - `src/App.tsx` (placeholder), `src/main.tsx`, `index.html`, `vite.config.ts`, `tsconfig.json`, `tsconfig.node.json`, `vite-env.d.ts`.
   - Existing bootstrap files (`.gitignore`, `README.md`, `package.json`, `.aiox/`) preserved ‚Äî the existing `package.json` must be replaced but the keywords, author (empty), license already set should roll forward.

2. **TypeScript strict mode** ‚Äî `tsconfig.json` has:
   ```jsonc
   {
     "compilerOptions": {
       "strict": true,
       "noUnusedLocals": true,
       "noUnusedParameters": true,
       "noFallthroughCasesInSwitch": true,
       "exactOptionalPropertyTypes": true
     }
   }
   ```
   Running `npx tsc --noEmit` or `npm run typecheck` passes.

3. **ESLint + Prettier configured** ‚Äî
   - `.eslintrc.cjs` (or flat `eslint.config.js`) with `@typescript-eslint/recommended`, `plugin:react-hooks/recommended`, `plugin:react/jsx-runtime`.
   - `.prettierrc` committed with project style (2-space indent, single quotes, trailing comma es5, no semicolons? ‚Äî team default: **WITH semicolons**, printWidth 100).
   - `npm run lint` passes on the scaffolded app.
   - `npm run format` formats the repo.

4. **Vitest installed** ‚Äî
   - `vitest`, `@vitest/ui`, `happy-dom`, `@testing-library/react`, `@testing-library/jest-dom` added.
   - `vitest.config.ts` committed (environment `happy-dom`, setup file loading `@testing-library/jest-dom`).
   - Sample test `src/App.test.tsx` renders `<App />` and asserts on placeholder text ‚Äî passes via `npm test`.

5. **GitHub Actions CI workflow** ‚Äî `.github/workflows/ci.yml` runs on `push` to `main` and `pull_request` to `main`:
   - Sets up Node 20 with npm cache.
   - Runs `npm ci`, `npm run lint`, `npm run typecheck`, `npm run test:ci` (Vitest `--run`), `npm run build`.
   - Uploads `dist/` as artifact.
   - All steps pass on the main branch after merge.

6. **`npm run dev` opens placeholder** ‚Äî The dev server starts on `http://localhost:5173` and the `<App />` component renders a centered page with the text **"LegoB ‚Äî bootstrapping"** (replace the Vite template default). Plain styling; no Tailwind yet (Tailwind arrives in a later story).

7. **`.env.example` committed (PO Must-Fix #M2)** ‚Äî Repo-root file with exactly:
   ```
   # See docs/architecture/tech-stack.md for the pinned LDraw mirror (resolved in Story 1.3).
   VITE_LDRAW_CDN_BASE=

   # Optional Sentry DSN ‚Äî leave blank in MVP (Story 5.5).
   VITE_SENTRY_DSN=
   ```
   `.gitignore` already excludes `.env` so this is the only env-like file tracked.

8. **README updated** ‚Äî `README.md` gains a "Quickstart" section:
   ```md
   ## Quickstart

   ```bash
   npm install
   cp .env.example .env
   npm run dev   # http://localhost:5173
   npm test
   npm run build
   ```
   ```

9. **All scripts present in `package.json`**:
   ```json
   {
     "scripts": {
       "dev": "vite",
       "build": "tsc -b && vite build",
       "preview": "vite preview",
       "test": "vitest",
       "test:ci": "vitest --run",
       "typecheck": "tsc -b --noEmit",
       "lint": "eslint . --ext ts,tsx --report-unused-disable-directives --max-warnings 0",
       "format": "prettier --write ."
     }
   }
   ```
   Replace the placeholder `echo` scripts from the bootstrap `package.json`.

## Dev Notes

- **Do NOT install Tailwind, Radix, Zustand, Three.js, or idb in this story** ‚Äî those land in subsequent stories (1.2+). This story is intentionally minimal to keep the scaffold reviewable.
- **Keep `.gitignore` intact.** It already covers `node_modules/`, `dist/`, `.env*`.
- **Commit order suggestion:**
  1. Run Vite template into the project directory (may need `--overwrite-package-json`-style care ‚Äî alternate: scaffold to `/tmp/x` and copy files in).
  2. Customize `tsconfig.json` for strict mode.
  3. Add ESLint + Prettier configs.
  4. Add Vitest config + sample test.
  5. Write `.github/workflows/ci.yml`.
  6. Commit `.env.example` + README quickstart update.
  7. Verify `npm run dev`, `npm run lint`, `npm test`, `npm run build` all succeed locally.
- **Don't** run `gh repo create` in this story ‚Äî that's the user's responsibility when they complete `gh auth login`.
- **Don't** touch anything under `docs/`, `.aiox/`, or `public/` (those come with later stories).
